ルール解説
SECTION 01
勝利条件
モルックリケットは普通のモルックと何が違う?
モルックに陣取りの要素を組み合わせたゲーム。1ターンに2投まで投げられるのが基本ルールで、12ターン終了時に相手よりスコアが高いプレイヤーが勝利(同点の場合はオープンしたスキットル数が多い方が勝ち)。
12ターン待たずに決着がつくことはある?
あります。自分が全てのスキットル(6〜12番)をオープンしていて、かつ自分のスコアが相手と同点以上であれば、その場でゲームセット。
POINT
  • 1ターン2投、12ターン終了時にスコアが高い方が勝利
  • 同点ならオープン数が多い方が勝ち
  • 全オープン＋スコアが同点以上なら即ゲームセット
SECTION 02
オープンとクローズ
「オープン」と「クローズ」の条件は?
どのスキットルも合計2マークを入れると「オープン」――自分の得点源(陣地)として確保できる。反対にすでに相手がオープンしているスキットルに自分が2マークを入れると「クローズ」――そのスキットルを盤面から排除できる。
ゲーム画面にあるDOUBLE(2 MARKS)とは?
スキットルは「直当て」で当てると1投で2マーク扱いになる。つまりまだ1マークも無い状態からでも、直当てなら1投で即座にオープン(陣地化)できる。
クローズしたスキットルは二度と使えない?
使えない。クローズしたスキットルは盤面から完全に消滅し、以降は全員マークも加点もできなくなる。相手の大事な得点源をクローズで潰す――これがモルックリケット最大の妨害戦術。どのスキットルをオープンして陣地を広げ、どのスキットルをクローズして相手の陣地を消すか、その見極めが攻略の鍵。
POINT
  • 合計2マークで「オープン」(自分の陣地化)
  • 「直当て」なら1投で2マーク扱い、最短1投でオープン可能
  • クローズされたスキットルは盤面から消滅し誰も使えなくなる
SECTION 03
得点の取り方
オープン後にどうスコアを伸ばす?
自分がオープン済みのスキットルに当てれば、そのスキットルの数字がそのままスコアに加算される。さらに、オープン済みのスキットルに「直当て」を決めればスキットルの数字の2倍の点数が一気に入る。12番に直当てなら一気に24点。
まだ1マーク状態のスキットルに直当てを決めたら?
直当てで入る2マークのうち最初の1マークがオープンに消費され、残った1マーク分が通常の得点としてスコアに加算される。
POINT
  • オープン済みのスキットルに当てるとその数字がそのまま加点
  • オープン済みのスキットルへの「直当て」は数字の2倍の点数
  • 未オープンのスキットルへの「直当て」はオープン+通常得点を同時に獲得
SECTION 04
複数本当たった場合(MULTI HIT)
スキットルが複数本まとめて倒れたらどう処理する?
「MULTI HIT」として処理する。倒れたスキットルの中に自分がすでにオープンしているものが含まれていれば、倒れた本数の合計がそのままスコアに加算される。オープン済みの7・8・9番をまとめて倒したら3点加算。スキットルの数字ではなく本数がそのまま点数になる点に注意。
未オープンと相手オープン済みが混ざって当たった場合は?
いずれもまだ自分のものでない番に対しては、一律で1マークだけが同時に付与される。1投で複数のスキットルへ一気にマークを刻める布石になり、運が良ければ複数本まとめて一気にオープンも狙える。
POINT
  • オープン済みへのMULTIは本数の合計がそのまま加点(数字ではない)
  • 未オープンのスキットルは一律1マークが同時に付与される
  • 複数本が同時に1→2マークなら一気に複数オープンも狙える
SECTION 05
その他のルール
先攻1ターン目に出る特殊なアナウンスは何?
先攻プレイヤーの最初のターン(2投)だけは、どの数字を当ててもマーク処理は行われず、当てた本数がそのままダイレクトに合計得点として加算される特殊ルールが発動する。後攻プレイヤーの1ターン目からは、この特殊ルールは適用されず通常通りのマーク処理になる。
コートのサイズと、コート外に出たスキットルの扱いは?
基本は横6m × 縦8mのコートで、プレイヤーの技量に合わせて自由に調整可能。コート外に出たスキットルはコートのエンドライン上(境界線の上)に戻す。他のスキットルとの距離がモルック棒1本分未満の場合はモルック棒1本分の距離をあけてエンドライン上に配置する。
POINT
  • この特殊ルールは先攻の1ターン目だけ適用される(後攻は1ターン目から通常ルール)
  • コートサイズは横6m × 縦8mが基本、自由に調整可
  • コート外のスキットルはエンドライン上に戻す

勝つための3大セオリー
01数字の大きいスキットルから狙う
11番・12番といった大きな数字を先に狙うのが基本戦略。その後のフェーズを有利に進めやすくなります。
02近いスキットルから狙う
狙いやすい手前側のスキットルに確実に当ててマークを重ねることが重要な戦略です。確実に自分の陣地を拡大していく戦略です。
03同点以上にしてクローズを狙う
勝利の絶対条件は「スコアが相手と同点以上であること」です。大差で負けている状態ではクローズに望みを出しても点差は縮まりません。まず点差を縮めて、勝利に近づいてからクローズで攻めるのがセオリーです。
カスタムモードの魅力
設定次第で初心者からガチ勢まで全員が白熱できるゲームに仕立てられる、カスタム対戦の主な設定項目です。
3人・4人対戦への拡張(PLAYER COUNT) 2人だけの対戦だけでなく、最大4人での大乱戦が可能に。参加者全員がオープンするまでスキットルはクローズされないため、独走する一人をみんなで囲んで阻止する熱い駆け引きが生まれます。
マッチフォーマット(MATCH FORMAT) 1セット完結だけでなく、2本先取〜最大5本先取のセット制形式に対応。セット間の先攻・後攻の順番を自由に設定でき、本格的な試合を楽しめます。
連続ミス失格ルール(BURST SYSTEM) ONにすると、3〜6回(スライダーで自由設定)連続でMISS(0マーク)したプレイヤーがそのセットを失格。サバイバル仕様で、たった1回外すだけで終わるという極限の緊張感がゲームを引き立てます。
ハンデ設定(HANDICAP) ONにすると、プレイヤーごとに初期スコアや特定スキットルの初期マーク数を付与できます。初心者と上級者が混ざった試合でも実力差を補正し、全員が白熱できるゲームにできます。
ボーナスマーク設定(BONUS MARKS) 通常の「DOUBLE(×2)」に加え、「TRIPLE(×3)」ボーナスを切り替え可能。独自のルールと組み合わせは自由――プレイヤー独自の縛りを設けてゲームをカスタマイズするのも上級者の楽しみ方です。
投げ持ち本数の変更(THROWS / TURN) 1ターンに投げる本数を、「1本」「2本(標準)」「3本」から選択可能。3本にすると加点とマークが網のように飛び交う乱戦になります。
必要マーク数の変更(OPEN MARKS PER SKITTLE) スキットルがオープンするのに必要なマーク数を、標準の「2マーク」から「3マーク」に変更可能。陣地の取り合いに深みを持たせられます。すべてのスキットルに対して変更可能です。
アプリの使用方法
1
モード選択とプレイヤー登録
ノーマルまたはカスタム対戦を選択。カスタムでは自由にルールを変更できます。ルールを設定したら GAME START をタップして試合開始。
2
当てたスキットルの選択
モルックを投げてスキットル(6〜12番)を当てたら、画面中央の BOARD & INPUT 盤面からその数字エリアをタップして選択状態(枠が緑線に発光)にします。複数本当たった場合は、当てた分すべてタップしてください。
3
単発ヒットのマーク数を指定
狙ったスキットルを1本だけ当てた場合は、下部パネルの SINGLE 行から確定したいマーク数をタップして選択確定にします。
4
複数当たり・ミスは各専用ボタンへ
2本以上まとめて当たった場合は、対象ピンを選択した後に MULTI ブロックをタップします。1本も当たらなかった場合(ノーヒット)は、一番下の MISS ブロックを1タップするだけで自動記録されます。
5
入力ミスはいつでも取り消し可能
画面下部には ‹ UNDO ボタンが常時配置されています。押し間違いやカウントの食い違いがあっても、いつでも正確に1手前の状態に巻き戻せるので安心です。
